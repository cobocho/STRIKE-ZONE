## 5. 실행 방법

* git clone 후 npm install 해 주세요.
<br/>

* 프로젝트 Root 디렉토리에 .env 파일을 생성한 뒤, 아래 정보를 입력 및 저장하세요.
  * **MONGO_URL**="자신의 MongoDB 접속 주소"
  * **PORT**="로컬 서버 구동사용할 포트번호"

<br/>

* npm start 명령으로 실행합니다.
